Useful information.

Read the text below for the best user experience.

Dear summoner, on the left-hand side of your screen you can see the links section. Once you have started your adventure, the different choices will be displayed there. Make sure you choose wisely!

On the right-hand side you can see the Important section. As you go through the story a restart button will appear on every 3rd option you get. If you feel like you did not make the right choice, feel free to click it and it will take you back to the home page where you can start over.

Once you have finished the game, click the End Game button which will bring you back to the main page.

This Choose You Own Story game is inspired by one of the best games ever created - World of Warcraft. Some of the names in this story match the names used in World of Warcraft, but the story is fully unique. Make sure you check out the game here.

And last but not least, Enjoy!
